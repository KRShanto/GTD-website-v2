## Overview

A modern, responsive website for GTD Media Production featuring a comprehensive showcase of video production services and an intelligent AI chatbot assistant.

## Features

- **Responsive Design**: Optimized for all devices
- **AI Chatbot**: Intelligent assistant powered by OpenAI GPT-4
- **Service Showcase**: Complete overview of video production services
- **Client Testimonials**: Real client success stories
- **Interactive Gallery**: Video portfolio showcase
- **Contact Integration**: Seamless contact form with email functionality

## Environment Setup

1. Copy `.env.example` to `.env.local`
2. Add your OpenAI API key:
   ```
   OPENAI_API_KEY=your_openai_api_key_here
   ```
3. Get your API key from: [OpenAI Platform](https://platform.openai.com/api-keys)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables (see Environment Setup above)

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view the website
